# ProtoSolid

Cross-platform parametric CAD. Based on boundary representation model and euler operators.

Built with Rust & Electron + Vue.js


## Installation

  Install Rust & Node.js

  ```sh
  npm install
  ```

## Usage

  ### Run in a browser
  ```sh
  npm run dev
  ```

  ### Build as a standalone application
  ```sh
  npm run pack-electron
  ```
