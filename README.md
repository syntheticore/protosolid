# ProtoSolid

Cross-platform parametric CAD. Built with Rust & Electron + Nuxt


## Installation

  Install Rust & Node.js

  ```sh
  npm install
  ```

## Usage

  ### Debug in a browser
  ```sh
  npm run dev
  ```

  ### Build as a standalone application
  ```sh
  npm run pack-electron
  ```
